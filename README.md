# TeamNotes 

TeamNotes is a simple full-stack note-taking web application built to demonstrate
a **clean development workflow** and **industry-level DevOps practices**.

This repository is intentionally developed step-by-step, starting from a basic
application and gradually evolving into a fully Dockerized and CI/CD-enabled
project.

---

##  Project Goals

- Build a simple but real-world usable web application
- Follow clean project structure from day one
- Apply DevOps practices the same way companies do
- Keep the setup understandable and reproducible

---

##  Tech Stack

### Application
- **Frontend:** React
- **Backend:** Node.js (Express)

### DevOps (added step by step)
- **Version Control:** Git & GitHub
- **Containerization:** Docker
- **CI/CD:** GitHub Actions
- **Code Quality:** SonarCloud
- **Security Scanning:** Trivy

---

---

##  Development Approach

This project is built incrementally:

1. Clean repository setup
2. Backend development
3. Frontend development
4. Dockerization (backend & frontend)
5. Multi-container orchestration
6. CI pipeline implementation
7. Code quality & security integration

Each step is committed separately to keep the history clean and readable.
